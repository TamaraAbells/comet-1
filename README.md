# Comet ([joincomet.app](https://joincomet.app))

All-in-one chat and forums for communities.

## Tech Stack

### Frontend
- React.js
- Next.js
- Tailwind CSS
- React Query

### Backend
- Node.js
- TypeScript
- PostgreSQL
- Express.js
- Apollo GraphQL Server
- TypeGraphQL
- TypeORM
